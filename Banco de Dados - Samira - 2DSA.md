# **Banco de Dados - Samira - 2DSA**





#### **O que é um Projeto de Banco de Dados?**

É o planejamento detalhado e pensado da criação de uma estrutura de banco de dados para armazenar e organizar os dados de maneira útil. 



Ela contém algumas fases cruciais que são: 

* Definir os requisitos e necessidades do projeto;
* Construir uma representação visual (diagramas) de como funcionará;
* Como organizar todas as informações;
* Quais ferramentas, SGBD e linguagem usar.



Elas são úteis para a criação do projeto para funcionar da melhor forma com uma gestão fluida e uma manutenção fácil e intuitiva. 



#### 

#### **O que é um Sistema de Banco de Dados?**

#### 

É um sistema de armazenamento de dados/informações que usa o modelo CRUD (create, read, update e delete) além de processar e interpretar dados de maneira lógica e eficiente. São usados para projetos em diversos segmentos e são essenciais no meio tecnológico.



As principais linguagens usadas para programação Back-End (parte lógica e de dados) são:

* Java;
* PHP;
* Python;
* Node.js;



O SQL (Banco de dados Relacional, em tabelas) é usado para inserir dados, fazer consultas, atualizar e excluir dados. Contudo, o NoSQL (Banco de dados Não Relacional, com documentos, grafos ou chave-valor) vem conquistando lugar no mercado. Elas são usadas para "conversar" com o Banco.



Os SGBDs são os mediadores entre os usuários e os dados armazenados no Banco de Dados. São exemplos de SGBDs: 

* MySQL(relacional);
* PostgreSQL (relacional);
* Oracle Database (sistema relacional da Oracle);
* MongoDB (não relacional).
